# github 사용하기
1. 우측 상단 + 버튼 누르기
2. `+` 메뉴의 create repository 버튼 누르기
3. 'Repository name'란에 이름 설정 후 우측 하단의 create repository 누르기

4. git remote add origin https://github.com/echo153/TIL.git 복사
5. vscode terminal에 붙여넣기 
6. `git push -u origin` 입력
